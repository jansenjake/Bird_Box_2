Route Description

Method Path Description
GET / Renders the home page (index.html)
POST /login Allows users to log into their account .
GET /users Returns an array of all users registered .
GET /users/:id Returns an object of the user with matching ID .
PUT /users/:id Updates details of the user with the matching ID .
POST /register Encrypts password given by user, saves user database, and returns a JSON web token to the frontend.
DELETE /users/:id Removes user from the database .
GET /products Returns an array of all products .
GET /products/:id Returns an object of the selected product with matching ID .
POST /products Successfully saves product in relevant database .
PUT /products/:id Updates product in relevant database successfully -- only users with admin roles can update products .
DELETE /products/:id Deletes product from relevant database successfully -- only users with admin roles can update products .
